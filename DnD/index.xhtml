Drag'n Drop & Firefox 3.6

See related blog post on hacks.mozilla.org.

Supported objects: image, video (*.ogv), audio (*.ogg), text files (*.txt), text pieces (from a web page or an applicatoin), tabs, bookmarks, ... (from desktop or other pages).

To take full advantage of the demo, you need to run a modern browser. Try the latest beta of Firefox 3.6 (download).

Demo by Paul Rouget.

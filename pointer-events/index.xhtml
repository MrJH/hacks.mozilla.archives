CSS pointer-events & Firefox 3.6

See related blog post on hacks.mozilla.org.

This blue box is a replica of the tag list from twitter.com homepage. As you can see, there is a fade on the right part of the box. It's done via a div which is on top of the list, with a CSS Gradient Background. The tags are links (<a/>), so clickable. But the tags under the fade are not, because the div on top "stops" the mouse events (you don't click on a link you click on the fade box).

To avoid this behavior and have the tags clickable, you can make the fade div "click-through", via the pointer-events: none; CSS property. Enable this behavior by clicking the checkbox.
